## Знакомство
Поздравляю Вы стали участником "ПРОЕКТА Х"
Теперь вы один из подопотных кроликов :>, лан шучу

Данный проэкт представляет из себя систему автоматического тестирования на данных других пользователей,
таких же студентов, как и вы

Другими словами, программа компилит ваш файл лабораторной работы и прогоняет тесты на базе тестов других людей

## Инструкция
- Запушить файлы лабораторной работы
- Тесты прогоняются при каждом пуше (ветка main - надо ручками)

## Инструкция если ручками
- Зайти в "Actions"
- Создать новое действие в "Notify..."
- Посмотреть выполнение своих тестов в "Big data"

## Примечания
1. Пушить можно из любой ветки
2. Чтобы система работала, ваша задача залить всё в ветку main, а если нужно что-то отдельно потестить - используйте ветки
