# Информация

## Знакомство
Поздравляю Вы стали участником "ПРОЕКТА Х"
Теперь вы один из подопотных кроликов :>, лан шучу

Данный проэкт представляет из себя систему автоматического тестирования на данных других пользователей,
таких же студентов, как и вы

Другими словами, программа компилит ваш файл лабораторной работы и прогоняет тесты на базе тестов других людей

## Инструкция общая
- У вас должен появится свой репозиторий: student-....
- Клонируйте репозиторий на локалку или можете воспользоваться веб-интерфесом (для клонирования репозитория потребуется токен: Настройки -> Настройки разработчика -> Personal token -> generate token(classic)
- Загрузите файлы лабораторной работы
- Запушите файлы лабораторной работы на свой репозиторий 
- Тесты прогоняются при каждом пуше (ветка main - надо ручками)
- Посмотреть выполнение своих тестов в "Actions" -> "Big data test" -> run-tests

## Инструкция если ручками
- Всё как в инструкции выше до пункта "Тесты прогоняются...."
- Зайти в "Actions"
- Создать новое действие в "Notify..." -> "run workflows"
- Посмотреть выполнение своих тестов в "Big data" -> run-tests

## Примечания
1. Пушить можно из любой ветки
2. Чтобы система работала, ваша задача залить всё в ветку main, а если нужно что-то отдельно потестить - используйте ветки
